A WooCommerce addon plugin for WordPress that enables bulk discount settings, allowing customers to receive greater discounts as they purchase more.

== Description ==
##Buy more, Save more

The **Opal Upsale Quantity for WooCommerce** plugin for WordPress is a powerful tool designed to incentivize customers to purchase in larger quantities by offering progressive discounts. With this plugin, store owners can easily set up discount tiers based on the quantity of items a customer adds to their cart. The more products a customer buys, the higher the discount they receive. This feature is particularly useful for boosting sales and encouraging bulk purchases, making it a valuable addition to any WooCommerce store looking to increase average order value and improve customer satisfaction. The plugin is easy to configure, flexible, and integrates seamlessly with existing WooCommerce functionalities.

## Features
* Choosing display position of the discount badge on single product
* Custom quantities input 
* Show/Hide discount percent in cart and order
* Discount base on number of product items or cart items
* Using Shortcode <code>[ouqw]</code> to show badge discount of product
* Settings rules for product, category, product type ... or some special products
* Settings quantities range discount Percent
* Import/Export settings

## Benefits:
* **Increased Sales Volume:** By offering discounts based on quantity, the plugin encourages customers to buy more items, leading to higher overall sales.
* **Enhanced Customer Loyalty:** Customers are more likely to return to a store that offers value through bulk discounts, fostering long-term loyalty.
* **Boosted Average Order Value:** As customers are incentivized to purchase more to receive greater discounts, the average order value increases, benefiting your store's revenue.
* **Easy Configuration:** The plugin is user-friendly and easy to set up, allowing store owners to create and manage discount tiers with minimal effort.
* **Improved Inventory Management:** Encouraging bulk purchases can help reduce excess inventory, making it easier to manage stock levels and reduce holding costs.
* **Seamless WooCommerce Integration:** The plugin integrates smoothly with WooCommerce, ensuring that all discount settings and calculations are automatically applied during checkout without any compatibility issues.

== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/' directory of your WordPress site.
3. Activate plugin 'Opal Upsale Quantity for Woocommerce' from Plugins page

== Frequently Asked Questions ==
= Can I use one rule for multiple products? =
Of course! You can attach the rule to one or more product categories, or simply attach it to each product.
